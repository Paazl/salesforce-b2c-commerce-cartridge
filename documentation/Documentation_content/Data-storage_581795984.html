Paazl SFCC cartridge : Data storage

Created by Thomas Frot, last modified on jan 09, 2019


Using the paazl cartridge, the following data will be stored within Commerce Cloud:  

  • REST API token in basket custom attribute
  • Selected shipping info in Shipment custom attribute
  • Selected shipping method name in Shipment custom attribute (after Order is Placed)
  • Replace Shipment shipping address with Paazl selected pickup point address (after Order is Placed and only in case of pickup point delivery)
  • State of order committed into paazl system or not is tracked via order custom attribute notSavedInPaazl
