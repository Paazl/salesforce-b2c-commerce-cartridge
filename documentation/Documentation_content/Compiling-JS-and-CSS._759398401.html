Paazl SFCC cartridge : Business Manager setup

Created by Thomas Frot on mrt 15, 2019


The Paazl JS and CSS files are part of the int_paazl_core cartridge. This cartridge is used by both SG and SFRA site and the JS and CSS of this cartridge are compiled the same way for both sites.

How to compile JS and CSS files of the int_paazl_core cartridge: 

  • Go to the top level of your project (just above the 'cartridges' folder).

  • Make sure that your are using our package.json and webpack.config.js committed in GitHub in the link_paazl repository.

  • Run the  command: 
    npm install
  • Run the commands: 
    npm run compile:js && npm run compile:scss

For Site Genesis:

  • First compile the JS and CSS file from app_storefront_core cartridge as explained there: SiteGenesis README.md
  • Then compile the JS and CSS file from int_paazl_core cartridge as explained at the beginning of this page

For SFRA:

  • First compile the JS and CSS file from app_storefront_base as explained there: SFRA README.md file
  • Then compile the JS and CSS file from int_paazl_core as explained at the beginning of this page
