Paazl SFCC cartridge : Cartridge installation and Metadata import

Created by Thomas Frot, last modified on jan 09, 2019


Step 1: Install the cartridge

The Paazl LINK cartridge can be used with either Storefront Reference Architecture (SFRA), and controller-based SiteGenesis.

  • If you are using SFRA, install the cartridges int_paazl_sfra and int_paazl_core using Commerce Cloud UX-studio.
  • If you are using a controller-based integration, install int_paazl_controllers and int_paazl_core cartridges using Commerce Cloud UX-studio


Step 2: Import the metadata

To add new configuration items, import the predefined metadata: 

  1. Download the installation package on our GitHub page. 
  2. Open the package/site_import/sites/ folder.
  3. Rename the yourSiteId folder to the ID of your site in the Business Manager.
  4. Zip the site_import folder. 
  5. In the Business Manager, go to Administration > Site Development > Site Import & Export and import the zipped file.

After the import, attributes named paazl[attributeName] are added to: 

  • Administration > Site Development > System Object Types > Site Preferences > Attribute Definitions
  • Administration > Site Development > System Object Types > Shipment > Attribute Definitions
  • Administration > Site Development > System Object Types > Basket > Attribute Definitions

Also, the following services are added to Administration > Operations > Services:

  • service.paazl.rest.gettoken
  • service.paazl.rest.getSelectedOption
  • service.paazl.orderrequest.address
  • service.paazl.rest.commitOrder

Also, the following shipping method is added to [yourSiteId] > Merchant Tools  > Ordering >  Shipping Methods:

  • paazl

Also, the following job is added to Administration > Operations > Jobs

  • Paazl-Commit-Order
