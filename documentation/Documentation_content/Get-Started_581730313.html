Paazl SFCC cartridge : Get Started

Created by Thomas Frot, last modified on jan 09, 2019



Before you begin to integrate Commerce Cloud with the Paazl platform, make sure that you have performed the following steps:
  • Apply for a test account with Paazl.
  • Apply for a live account with Paazl.
  • Get your API key and API secret from both accounts.
  • In your Test and Live account, configure shipping setting as needed for your site.
  • Have a look to the Paazl Widget documentation here.
