Salesforce Commerce Cloud

Use Paazl's cartridge to offer easy and complete shipping solution



The Paazl LINK cartridge is a full shipping solution for your Salesforce Commerce Cloud ecommerce site. One integration gives you access to all features of the Paazl platform. 


The cartridge can be used with:

  • Commerce Cloud's Storefront Reference Architecture (SFRA) 
  • a controller-based SiteGenesis
