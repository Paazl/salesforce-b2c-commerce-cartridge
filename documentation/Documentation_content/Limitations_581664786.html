Paazl SFCC cartridge : Limitations

Created by Thomas Frot, last modified on dec 06, 2018


Paazl functionalities not available out of the box with the int_paazl cartridge:

  • You cannot connect to Paazl REST API to fetch available shipping option.
  • You cannot commit your order into Paazl Platform to be able to monitor it via your Paazl account (feature coming in the next release)

Using the Paazl shipping widget over the default Commerce Cloud shipping solution, the following SFRA and/or SiteGenesis features aren't supported:

  • Possibility to select shipping option from the Cart page (SFRA)
  • Multi shipping (SFRA and SiteGenesis)
